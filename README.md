# PDF Utility Pro

オフラインで動作する、高セキュリティなPDF変換・編集ツールです。
外部サーバーへデータを送信しないため、機密情報を含むPDFも安心して処理できます。

## 主な機能

- **PDFの画像化 (JPEG変換)**
  - PDFを画像に変換します。変換後の画像に透かし（ウォーターマーク）を追加可能です。
- **PDFのWord化**
  - PDF内のテキストを抽出し、Wordで編集しやすい形式に変換します。
- **PDF結合**
  - 複数のPDFファイルを一つにまとめます。
- **PDFセキュリティ保全**
  - PDF内のメタデータや個人情報を削除・クリーンアップし、安全な状態にします。

## 使い方

1. このツールはブラウザ上で動作します。
2. インターネット接続がなくても動作するため、オフライン環境で安全にご利用いただけます。
3. 処理したいファイルを画面上にドラッグ＆ドロップして操作を開始してください。

## 技術スタック
- Vite
- JavaScript / CSS
- PWA (Progressive Web App) 対応

## プライバシー・セキュリティについて
本アプリはクライアントサイドのみで処理が完結します。変換処理中にファイルが外部サーバーへ送信されることは一切ありません。

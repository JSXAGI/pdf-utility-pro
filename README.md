PDF Utility Pro (v2.0)
ブラウザのみで完結する、高セキュリティなローカルPDF変換・編集ツール。
サーバー通信を一切行わないため、機密情報を含むPDFもローカル環境で安心して処理可能です。

主な機能
🖼️ 画像変換 & ロゴ合成

PDFの各ページを画像に変換。ロゴ画像を設定すれば、全ページに自動合成されます。

📝 高品質Word抽出

PDF内のテキストを座標解析し、レイアウトを保持したままWord(.docx)へ変換します。

🔗 PDF結合

複数のPDFを順序立てて1つのファイルに統合します。

🛡️ 匿名化 (メタデータクリーナー)

PDFのプロパティ（作成者・タイトル等）を削除し、安全な状態にクリーンアップします。

使い方
本ツールはブラウザ上で直接動作する「オフラインファースト」なアプリです。

一度読み込めば、インターネット接続が切断された環境でも安定して動作します。

画面上の各メニューから対象ファイルを選択し、処理を開始してください。

技術仕様
Frameworkless Architecture: ビルドツール（Vite等）を排除した、ブラウザネイティブなES Modules設計。

Core Libraries:

pdfjs-dist (PDF解析)

pdf-lib (PDF操作)

docx (Word生成)

jszip (アーカイブ生成)

PWA (Progressive Web App): Service Workerによるオフライン対応と、アプリとしてのインストール機能を完全実装。

プライバシー・セキュリティについて
本アプリは「クライアントサイド（お使いのブラウザ内）」での処理を徹底しています。ファイルデータが外部サーバーへ送信されることは物理的にありえません。 あなたのデータは、あなたのデバイスの外に出ることはありません。
